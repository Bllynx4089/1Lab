Название проекта: Вычисление параметров геометрических фигур
Студенты: Кучаева Яна, Мещеряков Степан, группа АБс-524
Репозиторий: github.com/Bllynx4089/1Lab

Описание:
Данный проект представляет собой программу на C++, предназначенную для расчета параметров геометрических фигур по заданным исходным данным. В рамках первой лабораторной работы по дисциплине 
«Программирование» реализованы функции для работы с прямоугольником и трапецией. Код структурирован с использованием отдельных заголовочных файлов (rectangle.h, trapezoid.h) и соответствующих 
файлов реализации, что демонстрирует применение модульного подхода. Проект ведется с использованием системы контроля версий Git и размещен на платформе GitHub.
